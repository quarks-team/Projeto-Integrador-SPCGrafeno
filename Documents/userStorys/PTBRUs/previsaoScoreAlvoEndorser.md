# História de Usuário: Previsão de Pontuação Alvo - Endossante

### - [English - Version](https://github.com/quarks-team/Projeto-Integrador-SPCGrafeno/blob/main/Documents/userStorys/ENUs/previsaoScoreAlvoEndorser.md)

## Descrição
Como um usuário endossante, quero descobrir quais mudanças são necessárias para atingir uma pontuação alvo que fornecerei, para que eu possa tomar decisões informadas e melhorar minha situação de crédito.

## Definição de Pronto (DoR)
- [ ] A funcionalidade para calcular as mudanças necessárias está claramente definida.
- [ ] Os requisitos para os parâmetros de entrada da pontuação alvo estão documentados.
- [ ] O banco de dados contém informações suficientes para calcular as mudanças necessárias.

## Definição de Pronto (DoD)
- [ ] O usuário pode inserir uma pontuação alvo e receber sugestões de mudanças.
- [ ] As sugestões são precisas e compreensíveis.
- [ ] A funcionalidade foi testada e documentada.
- [ ] O feedback do usuário é coletado e analisado.

## Critérios de Aceite

### Entrada e Sugestões:
- [ ] O usuário pode inserir uma pontuação alvo e receber feedback com sugestões para melhorias.

### Sugestões Baseadas em Dados:
- [ ] As sugestões fornecidas são baseadas em dados reais e relevantes.

### Responsividade do Sistema:
- [ ] O sistema é responsivo e fornece resultados de forma oportuna.
